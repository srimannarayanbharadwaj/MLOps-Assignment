# Phase 3 MLflow Screenshots

Run training:

```powershell
python -m src.train
```

Open MLflow UI:

```powershell
mlflow ui
```

Then open the local URL shown in the terminal, usually:

```text
http://127.0.0.1:5000
```

Capture screenshots of:

- The `heart-disease-classification` experiment run comparison table.
- A model run detail page showing logged parameters and metrics.
- The artifacts section showing the confusion matrix and model artifact.
